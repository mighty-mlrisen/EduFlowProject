# Тестирование: Профиль и статистика

Модуль проверяет просмотр и редактирование профиля пользователя, загрузку аватара и отображение статистики.

---

## TC-PROF-01 — Просмотр профиля другого пользователя

| Поле | Описание |
|---|---|
| **ID** | TC-PROF-01 |
| **Название** | Просмотр публичного профиля другого пользователя |
| **Предусловие** | В системе зарегистрирован пользователь с опубликованными статьями; текущий пользователь может быть как авторизован, так и нет |
| **Шаги** | 1. Перейти на страницу профиля другого пользователя (через поиск или карточку автора статьи) |
| **Ожидаемый результат** | Отображаются: имя, аватар, биография, счётчики публикаций и подписчиков, список опубликованных статей; черновики автора не видны; кнопка «Подписаться» доступна для авторизованных пользователей |
| **Статус** | Пройден |

---

## TC-PROF-02 — Редактирование имени и биографии

| Поле | Описание |
|---|---|
| **ID** | TC-PROF-02 |
| **Название** | Редактирование имени пользователя и биографии в профиле |
| **Предусловие** | Пользователь авторизован; открыта страница редактирования своего профиля |
| **Шаги** | 1. Изменить имя пользователя на «Новое Имя» <br> 2. Ввести биографию (до 300 символов) <br> 3. Нажать «Сохранить» |
| **Ожидаемый результат** | Данные обновлены в базе данных; на странице профиля отображаются новые имя и биография; имя пользователя обновилось в шапке сайта (AppHeader) без перезагрузки страницы |
| **Статус** | Пройден |

---

## TC-PROF-03 — Загрузка аватара

| Поле | Описание |
|---|---|
| **ID** | TC-PROF-03 |
| **Название** | Загрузка и обновление аватара пользователя |
| **Предусловие** | Пользователь авторизован; подготовлен файл изображения в формате JPG или PNG размером до 5 МБ |
| **Шаги** | 1. Перейти на страницу редактирования профиля <br> 2. Нажать на область аватара или кнопку «Изменить фото» <br> 3. Выбрать файл изображения <br> 4. Сохранить профиль |
| **Ожидаемый результат** | Изображение загружено в Supabase Storage; URL аватара обновлён в профиле пользователя; новый аватар отображается на странице профиля и в шапке сайта |
| **Статус** | Пройден |

## TC-PROF-04 — Персональная статистика автора

| Поле | Описание |
|---|---|
| **ID** | TC-PROF-04 |
| **Название** | Просмотр персональной статистики автора |
| **Предусловие** | Пользователь авторизован; у него есть минимум одна опубликованная статья с лайками и комментариями |
| **Шаги** | 1. Перейти в раздел «Статистика» в своём профиле |
| **Ожидаемый результат** | Отображаются: общее количество опубликованных статей, суммарное количество лайков, суммарное количество комментариев, количество подписчиков; данные соответствуют фактическому состоянию записей в базе |
| **Статус** | Пройден |

---

## TC-PROF-05 — Общая статистика платформы

| Поле | Описание |
|---|---|
| **ID** | TC-PROF-05 |
| **Название** | Отображение агрегированной статистики всей платформы |
| **Предусловие** | На платформе есть зарегистрированные пользователи, опубликованные статьи и активность (лайки, комментарии) |
| **Шаги** | 1. Перейти на страницу общей статистики платформы |
| **Ожидаемый результат** | Отображаются агрегированные показатели: общее количество пользователей, статей, лайков, комментариев; цифры соответствуют реальным данным БД; страница доступна как авторизованным, так и гостевым пользователям |
| **Статус** | Пройден |
