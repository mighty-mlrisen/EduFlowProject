# Тестирование: Лента и поиск

Модуль проверяет отображение главной ленты, персональной ленты подписок, фильтрацию по категориям и поиск по контенту.

---

## TC-FEED-01 — Отображение главной ленты статей

| Поле | Описание |
|---|---|
| **ID** | TC-FEED-01 |
| **Название** | Отображение главной ленты опубликованных статей |
| **Предусловие** | В системе есть опубликованные статьи; пользователь открыл главную страницу |
| **Шаги** | 1. Открыть главную страницу приложения <br> 2. Наблюдать за отображением ленты |
| **Ожидаемый результат** | Список статей загружен и отображён в порядке убывания даты публикации; каждая карточка содержит заголовок, автора, категорию, дату и счётчик лайков; черновики и удалённые статьи не отображаются |
| **Статус** | Пройден |

---

## TC-FEED-02 — Фильтрация статей по категории

| Поле | Описание |
|---|---|
| **ID** | TC-FEED-02 |
| **Название** | Фильтрация ленты по выбранной категории |
| **Предусловие** | В системе есть статьи как минимум в двух разных категориях |
| **Шаги** | 1. Открыть главную ленту <br> 2. Перейти в раздел «Категории» <br> 3. Выбрать конкретную категорию (например, «Backend») |
| **Ожидаемый результат** | Отображаются только статьи выбранной категории; статьи других категорий скрыты; заголовок страницы содержит название выбранной категории |
| **Статус** | Пройден |

---

## TC-FEED-03 — Персональная лента из подписок

| Поле | Описание |
|---|---|
| **ID** | TC-FEED-03 |
| **Название** | Персональная лента статей авторов из подписок |
| **Предусловие** | Пользователь авторизован и подписан на не менее чем одного автора, у которого есть опубликованные статьи |
| **Шаги** | 1. Перейти в раздел «Мои подписки» / «Лента подписок» |
| **Ожидаемый результат** | Отображаются только статьи авторов, на которых подписан пользователь; статьи других авторов отсутствуют; порядок — по дате публикации (новые первыми) |
| **Статус** | Пройден |

---

## TC-FEED-04 — Поиск статей по ключевому слову

| Поле | Описание |
|---|---|
| **ID** | TC-FEED-04 |
| **Название** | Глобальный поиск статей по ключевому слову |
| **Предусловие** | В системе опубликована статья, заголовок или текст которой содержит слово «Docker» |
| **Шаги** | 1. Нажать на иконку поиска или перейти на страницу поиска <br> 2. Ввести запрос «Docker» <br> 3. Нажать Enter или кнопку поиска |
| **Ожидаемый результат** | Отображаются статьи, содержащие слово «Docker» в заголовке или теле; нерелевантные статьи не отображаются; результаты сгруппированы или помечены как «Статьи» |
| **Статус** | Пройден |

---

## TC-FEED-05 — Поиск по имени автора

| Поле | Описание |
|---|---|
| **ID** | TC-FEED-05 |
| **Название** | Поиск пользователей по имени |
| **Предусловие** | В системе зарегистрирован пользователь с именем «Иван Иванов» |
| **Шаги** | 1. Перейти на страницу поиска <br> 2. Ввести запрос «Иван» |
| **Ожидаемый результат** | В результатах поиска в разделе «Авторы» отображается карточка пользователя «Иван Иванов» с аватаром и кнопкой перехода на профиль; пользователи без совпадения по имени не отображаются |
| **Статус** | Пройден |
