# Тестирование: Статьи и редактор

Модуль проверяет создание, редактирование, удаление статей и функциональность WYSIWYG-редактора TipTap.

---

## TC-ART-01 — Создание и публикация новой статьи

| Поле | Описание |
|---|---|
| **ID** | TC-ART-01 |
| **Название** | Создание и публикация новой статьи |
| **Предусловие** | Пользователь авторизован; открыта страница создания статьи |
| **Шаги** | 1. Ввести заголовок статьи <br> 2. Выбрать категорию из выпадающего списка <br> 3. Написать текст в редакторе (минимум 1 абзац) <br> 4. Нажать «Опубликовать» |
| **Ожидаемый результат** | Статья сохранена в базе данных со статусом `PUBLISHED`; пользователь перенаправлен на страницу статьи; статья отображается в общей ленте |
| **Статус** | Пройден |

---

## TC-ART-02 — Сохранение статьи как черновик

| Поле | Описание |
|---|---|
| **ID** | TC-ART-02 |
| **Название** | Сохранение статьи как черновик |
| **Предусловие** | Пользователь авторизован; открыта страница создания статьи |
| **Шаги** | 1. Ввести заголовок и частичный текст статьи <br> 2. Нажать «Сохранить как черновик» |
| **Ожидаемый результат** | Статья сохранена со статусом `DRAFT`; черновик виден только автору в его профиле; в общей ленте статья не отображается |
| **Статус** | Пройден |

---

## TC-ART-03 — Редактирование опубликованной статьи

| Поле | Описание |
|---|---|
| **ID** | TC-ART-03 |
| **Название** | Редактирование опубликованной статьи автором |
| **Предусловие** | Авторизованный пользователь является автором существующей опубликованной статьи |
| **Шаги** | 1. Открыть страницу своей статьи <br> 2. Нажать кнопку «Редактировать» <br> 3. Изменить заголовок и/или текст <br> 4. Нажать «Сохранить изменения» |
| **Ожидаемый результат** | Обновлённые данные статьи отображаются на её странице; дата изменения обновлена; статус `PUBLISHED` сохранён |
| **Статус** | Пройден |

---

## TC-ART-04 — Удаление статьи

| Поле | Описание |
|---|---|
| **ID** | TC-ART-04 |
| **Название** | Удаление статьи автором |
| **Предусловие** | Авторизованный пользователь является автором существующей статьи |
| **Шаги** | 1. Открыть страницу своей статьи <br> 2. Нажать кнопку «Удалить» <br> 3. Подтвердить удаление в диалоге |
| **Ожидаемый результат** | Статья удалена из базы данных; пользователь перенаправлен на главную страницу; статья больше не отображается в лентах и поиске; кэш саммари в Redis инвалидирован |
| **Статус** | Пройден |

## TC-ART-05 — Вставка блока кода в редакторе

| Поле | Описание |
|---|---|
| **ID** | TC-ART-05 |
| **Название** | Вставка и отображение блока кода через редактор TipTap |
| **Предусловие** | Пользователь авторизован; открыт редактор статьи |
| **Шаги** | 1. Установить курсор в теле статьи <br> 2. Нажать кнопку «Блок кода» на панели инструментов редактора <br> 3. Ввести код (например, фрагмент на Python) <br> 4. Опубликовать статью и открыть её страницу |
| **Ожидаемый результат** | Блок кода отображается с моноширинным шрифтом и подсветкой синтаксиса; форматирование сохранено корректно при сохранении и повторном открытии |
| **Статус** | Пройден |
