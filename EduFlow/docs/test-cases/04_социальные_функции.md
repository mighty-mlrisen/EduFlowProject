# Тестирование: Социальные функции

Модуль проверяет лайки, комментарии, подписки на авторов и добавление статей в избранное.

---

## TC-SOC-01 — Лайк статьи

| Поле | Описание |
|---|---|
| **ID** | TC-SOC-01 |
| **Название** | Постановка и снятие лайка со статьи |
| **Предусловие** | Пользователь авторизован; открыта страница статьи другого автора |
| **Шаги** | 1. Нажать кнопку «Лайк» на странице статьи <br> 2. Убедиться, что счётчик лайков увеличился на 1 <br> 3. Нажать кнопку «Лайк» повторно |
| **Ожидаемый результат** | Шаг 2: счётчик лайков увеличивается на 1; кнопка меняет визуальное состояние (активна). Шаг 3: лайк снят, счётчик уменьшается на 1; кнопка возвращается в неактивное состояние. Повторные нажатия не создают дублирующих записей |
| **Статус** | Пройден |

---

## TC-SOC-02 — Добавление комментария к статье

| Поле | Описание |
|---|---|
| **ID** | TC-SOC-02 |
| **Название** | Добавление комментария авторизованным пользователем |
| **Предусловие** | Пользователь авторизован; открыта страница любой опубликованной статьи |
| **Шаги** | 1. Перейти в раздел комментариев на странице статьи <br> 2. Ввести текст комментария в поле ввода <br> 3. Нажать «Отправить» |
| **Ожидаемый результат** | Комментарий появляется в списке под статьёй; отображается имя автора комментария, текст и время публикации; поле ввода очищается после отправки |
| **Статус** | Пройден |

---

## TC-SOC-03 — Подписка на автора и отписка

| Поле | Описание |
|---|---|
| **ID** | TC-SOC-03 |
| **Название** | Подписка на автора и последующая отписка |
| **Предусловие** | Пользователь авторизован; открыт профиль другого пользователя, на которого нет подписки |
| **Шаги** | 1. Нажать кнопку «Подписаться» на странице профиля <br> 2. Убедиться, что счётчик подписчиков на профиле автора увеличился <br> 3. Нажать кнопку «Отписаться» |
| **Ожидаемый результат** | Шаг 1: подписка оформлена; кнопка меняется на «Отписаться»; автор появляется в персональной ленте подписок. Шаг 3: подписка отменена; счётчик уменьшается; статьи автора пропадают из ленты подписок |
| **Статус** | Пройден |

---

## TC-SOC-04 — Добавление статьи в избранное

| Поле | Описание |
|---|---|
| **ID** | TC-SOC-04 |
| **Название** | Сохранение статьи в избранное и просмотр списка |
| **Предусловие** | Пользователь авторизован; открыта страница опубликованной статьи |
| **Шаги** | 1. Нажать кнопку «Сохранить» (иконка закладки) на странице статьи <br> 2. Перейти в раздел «Избранное» в профиле |
| **Ожидаемый результат** | Шаг 1: иконка закладки меняет состояние на «сохранено». Шаг 2: статья присутствует в списке избранного с заголовком, автором и датой сохранения; при повторном нажатии на закладку статья удаляется из избранного |
| **Статус** | Пройден |

---

## TC-SOC-05 — Удаление собственного комментария

| Поле | Описание |
|---|---|
| **ID** | TC-SOC-05 |
| **Название** | Удаление комментария его автором |
| **Предусловие** | Пользователь авторизован; у него есть ранее оставленный комментарий под статьёй |
| **Шаги** | 1. Открыть страницу статьи с комментарием <br> 2. Найти свой комментарий <br> 3. Нажать кнопку «Удалить» рядом с комментарием <br> 4. Подтвердить удаление |
| **Ожидаемый результат** | Комментарий удалён из списка; другие комментарии не затронуты; кнопка «Удалить» не отображается у комментариев других пользователей |
| **Статус** | Пройден |
